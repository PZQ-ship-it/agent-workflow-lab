# Initial Brief

Date: 2026-06-24
Status: seed note

## Motivation

Agentic coding workflows are changing quickly. People are experimenting with skills, harnesses, loops, subagents, MCP tools, hooks, and repo-specific operating manuals. Many examples are useful, but the useful part is often hidden behind a catchy label.

This repository exists to make those patterns inspectable and testable.

## Scope

In scope:

- Public examples of Codex, Claude, and adjacent agentic coding workflows.
- Local experiments that use public or synthetic inputs.
- Short notes that separate observed behavior from personal judgment.
- Small harnesses that can be rerun.

Out of scope:

- Private user transcripts.
- Secrets, credentials, cookies, API keys, or local account data.
- Full mirrors of third-party docs or paid content.
- Hype-only lists without testable claims.

## First Milestones

- Build a vocabulary map for `skill`, `harness`, `loop`, `agent`, `hook`, and `MCP`.
- Collect 10 public examples with source links and one-line claims.
- Run 3 small experiments:
  - one skill-style workflow,
  - one completion/review loop,
  - one harness or eval wrapper.
- Write a short comparison note on when each pattern is worth using.
